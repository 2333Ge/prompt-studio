我想要开发一个提示词工具，主要功能是方便管理、存储查看prompt、交互输入便捷化、不负责AI交互执行，需要支持：
1. 本地存储，后续可能会拓展到远程，暂不考虑
2. 多版本历史（类似git的感觉）
    a. 历史记录存储（支持备注此次历史记录内容）
    b. 版本对比
    c. 一键回滚
3. 支持导出、导入
4. 支持模板变量，对于不同的变量可以支持不同的输入操作，比如图片可以上传、或者从图片库选择等，日期可以日历选择等，后续补充细节，比如
```
请帮我分析这张图片：

{{image}}

请用 {{language}} 输出。

输出风格：

{{style}}

今天日期：

{{date}}
```

```
然后单独维护一份 Schema，用于xrender 渲染
{
  "image": {
    "type": "image",
    "title": "图片",
    "required": true
  },
  "language": {
    "type": "select",
    "options": ["中文", "English", "日本語"],
    "default": "中文"
  },
  "style": {
    "type": "textarea"
  },
  "date": {
    "type": "date"
  }
}
```
可以切换到schema编辑模式，完善表单的交互形式

5. 支持翻译（注意一些内容不需要翻译，比如图片链接、代码等）
6. 支持命名、备注、分类、标签、评分、检索
    a. 其中标签支持二级分类，比如模型--deepseek这种
7. 支持分级系统，可以配置隐私状态下才能看的内容、隐私状态下才能导出对应内容，隐私状态开启要隐蔽一点，比如连点某个地方10下才出开关等
8. 支持存储对应 prompt 结果、所用模型（同模型可能不止一个结果需要存）

后续

9. 支持 iframe 页面小窗打开三方网站，方便进行联动操作，比如操作翻译等
